MRS リリースパッケージ

このディレクトリには、MRS製品群のバイナリ配布用パッケージが含まれています。

ディレクトリ構成

  +- mrs               MRSのコアAPI
  +- mrs_room          MRS Roomサーバ
  +- mrs_ws            MRSでWebSocketを扱う場合のサンプル
  +- VERSION.txt       技術サポートの時に使うバージョン記述ファイル

上記に含まれるそれぞれの製品の概要は、以下の通りです。

  • mrs : MRSのコアAPIです。C++用のライブラリ本体、ヘッダ、ドキュメント、サンプルコードを含みます。 ドキュメントは、 MRS_API_Overview.html を参照してください。
  • mrs_room : 「ルーム」への参加と脱退という概念を用いてマルチプレイゲームのデータ同期を行うための「ルームサーバ」 を実装するために必要なライブラリ、ヘッダ、ドキュメント、サンプルコードを含みます。 ルームサーバは、サンプルコードを拡張してゲームの専用サーバーを実装することを念頭に設計されています。 ドキュメントは、 ROOM_API_Overview.html を参照してください。
  • mrs_ws：MRSライブラリのWebSocket用のクライアントライブラリで、
    ブラウザで動作するJavaScriptのラッパーとサンプルコード、テストコードを含みます。
    ドキュメントは mrs_ws/documents/ ディレクトリに格納されています。

    全体の概要については、 MRS_WebSocket_Overview.html　を参照してください。

    サンプルゲームのソース解説は MRS_WebSocket_Samplegame.html　を参照してください。

    自動テストの方法については、 MRS_WebSocket_Testing.html　を参照してください。

製品の階層構造

MRS製品群は、以下のような階層構造をもっています。

        　　ゲームなどのアプリケーション
------------------------------------------------------
 mrs_room|pusher|ws などの上位ライブラリ/アプリケーション群
------------------------------------------------------
      MRS (Monobit Revolution Server) コアライブラリ
------------------------------------------------------
      libuv, OpenSSL などの依存ライブラリ
------------------------------------------------------
              ソケットAPI 
------------------------------------------------------
  LinuxやWindowsなどのオペレーティングシステム層

変更履歴は、ChangeLog.html を参照してください。
